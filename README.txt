Для запуска приложения использовался Tomcat v9 
ссылка для скачивания: https://tomcat.apache.org/download-90.cgi

-----------------------------------------------
Настройка конфигурации для запуска приложения:
Добавить конфигурацию -> Tomcat Server -> Local

В строке "Application Server" указать путь к распакованому серверу Tomcat

В разделе "Before launch" добавить артефакт "war:exploded"

В разделе "Open browser" в строке "URL" дописать "/login", что бы приложение при запуске стартовало с страницы авторизации
